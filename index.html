Boston Marathon Finisher Rates: Thoughness Index

This map was built using QGIS and Carto. The map characterizes statistics about finishers of the 2016 Boston Marathon by the participants home state. A toughness index has been calculated by multiplying the finisher rate (finishers/starters) by the rate of finishers from a given state population (finishers/state population).

  • See the map: 2016 Boston Marathon Finisher Rates: Toughness Index
  • Data Source: Boston Athletic Association
